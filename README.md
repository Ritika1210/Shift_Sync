# ShiftSync 🗓️

ShiftSync is a modern workforce scheduling web application built using React and Vite. It enables managers to efficiently create, manage, and monitor employee shifts with an intuitive and interactive interface.

---
 ## *LIVE DEMO  - https://shift-sync-iota.vercel.app/
     
## 🚀 Features

* 📅 Create and manage weekly employee shifts
* 🔄 Drag-and-drop functionality for easy scheduling
* 👥 Role-based views (Manager / Employee)
* ⚠️ Real-time conflict detection:

  * Overlapping shifts
  * Excessive working hours
  * Invalid scheduling patterns
* 📊 Dashboard with key insights:

  * Total employees
  * Total shifts
  * Total working hours
  * Violations detected
* 🧮 Live hour tracking per employee
* 📆 Interactive 7-day grid-based schedule UI
* 📄 Export schedules to PDF for sharing and reporting

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript
* **Build Tool:** Vite
* **Styling:** CSS
* **Other:** Drag-and-drop libraries, PDF export utilities
